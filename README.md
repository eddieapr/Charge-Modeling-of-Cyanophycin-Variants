# Charge-Modeling-of-Cyanophycin-Variants
MATLAB scripts for pH-dependent charge speciation modeling of cyanophycin granule peptide (CGP) and constituent amino acids (arginine, aspartic acid, lysine) to inform ion exchange-based amino acid recovery.

This repository contains MATLAB scripts developed to model the pH-dependent charge speciation of cyanophycin granule peptide (CGP) and its constituent amino acids as described in Apraku et al. (manuscript in preparation). The model applies Henderson-Hasselbalch-based fractional ionization to compute net charge as a function of pH for free amino acids and for n-length CGP polymer chains of varying lysine content. Scripts include charge speciation curves for arginine, aspartic acid, and lysine; intrinsic polymer charge models for insoluble (β-Asp-Arg) and soluble (β-Asp-Arg-Lys) CGP variants; chain-length convergence analysis (n = 1 to 10,000); and a solubility heuristic based on fractional charge thresholds. pKa values used in the model are provided in Table S2 of the associated manuscript.
